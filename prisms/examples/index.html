Welcome to Pristine!

This is a sample page for your new Pristine site. Let's start by learning how to changing the routes. No php knowledge necessary!
There are currently 2 json configuration files. One is called pdec.json and one is called mdec.json
To manipulate routes accessible from the web, edit pdec. For routes related to php includes (Modules), Edit Mdec.

Let's get started!

Open the route configuration file for sites (sdec.json)
Edit Landing->Pages->Home->Src from ./sites/examples/index.html to ./sites/landing/index.html
Now create a folder called landing inside of the prisms folder and then create an index.html within that.
Any traffic towards the root domain (ex: https://domain.com/) will now be redirected to the index.html (sites/landing/index.html). It's as easy as that!

Now update the site to see your changes.
